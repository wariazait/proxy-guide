# 🍏 Настройка VPN-клиентов для macOS

В этом разделе собраны инструкции по настройке современных VPN-клиентов для компьютеров Mac. Все представленные клиенты поддерживают процессоры как на базе **Intel**, так и **Apple Silicon (M1/M2/M3)**.

## Доступные клиенты

На данный момент я рекомендую использовать следующий клиент:

* **[v2rayN](./v2rayn.md)** — один из самых популярных и мощных клиентов, который теперь официально поддерживает macOS. Отлично работает с VLESS, Reality, Hysteria2 и поддерживает удобную настройку маршрутизации через региональные пресеты. **(Рекомендуемый выбор)**

> 🚧 *Инструкции для других клиентов находятся в разработке.*
