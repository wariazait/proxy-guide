# Настройка VPN-клиентов для Windows

В этом разделе собраны инструкции по настройке современных VPN-клиентов для операционных систем Windows 10 и Windows 11.

## Доступные клиенты

На данный момент я рекомендую использовать следующий клиент:

* **[v2rayN](./v2rayn.md)** — один из самых популярных и мощных клиентов для Windows. Отлично работает с VLESS, Reality, Hysteria2 и поддерживает удобную настройку маршрутизации через региональные пресеты. **(Рекомендуемый выбор)**
